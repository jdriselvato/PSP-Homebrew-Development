What is this?

A simple popstation GUI (as the title says). Popstation converts PSX games into EBOOT format, which is playable on the PSP.

How do you use this?

Note: make sure you have the file BASE.PBP in the Data folder.
Note 2: this assumes you already have a PSX Image - if you don't, see the Guide to playing PSOne/PSX games on the PSP section for more info
Okay, start up the program, then select your PSX Image file (must be an ISO, BIN, CUE, IMG, CCD or MDF file). It is recommended to select a CCD or CUE file if possible. After that, make sure the Game Title displays the correct value - if not, click the Select button (next to the Game Title field) and select your game. Now click Go and you're done :)

To convert an EBOOT back into an ISO, load the EBOOT, then click the Extract ISO from EBOOT button. Simple eh?

FAQ

How do I compress an already made EBOOT?
Open the EBOOT in the program, select Extract ISO from EBOOT then load the ISO in the PSX Image field, select your compression level, and click Go.

My game has odd graphic glitches etc!
You can try the PAL -> NTSC fixes. They may work, however, Simple Popstation GUI isn't a complex PAL -> NTSC fixer. Try looking for Zapper2000, and run your Image file through that.

I converted an ISO with the program, but when I try to play a game, all I get is a black screen!
Try dumping the disc using BIN or IMG format instead (don't try converting the ISO to BIN or IMG). Once you've done that, load the CCD or CUE file (they come along with the BIN/IMG images) into the program, and try converting.

I loaded a BIN file into the program, but when I try to play the game, my background music does not play!
Try adding the CUE file instead.
