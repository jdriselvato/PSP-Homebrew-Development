Classic 3d wireframe cube demo in LUA  for PSP by Nils - ventoline.com
v 1.0 __ 13/08/2005
Many thanks to Shine for LuaPlayer, this wonderful toy ! See luaplayer.org for more :)


Installation & Run notes:

1. Copy Both directories luaplayer and luaplayer% to PSP/GAME/
2. Go Game > Memory Stick > Lua Player
3. Use pad or analog pad to move the cube ... Press Start to exit, or press Select to print a screenshot to memory stick.
